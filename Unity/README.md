# VRDemos
Anwendungen, die mit Unity und den verschiedenen Plugins wie Steam, MiddleVR oder VRTK erstellt werden.

Das Verzeichnis helloVive enthält das aktuelle Hello World!, vewendet wird hier das Steam-Plugin.

Die Verzeichnisstruktur orientiert sich an den verwendeten Plugins:
- Cardboard enthält Projekte für Google Cardboad
- MVR enthält Projekte mit MiddleVR
- viveInput enthält Projekte mit der Vive Input Utility
- helloVive enthält das aktuelle Hello World Projekt (mit Steam).
