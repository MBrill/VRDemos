# Unity-Anwendungen mit MiddleVR

In diesem Verzeichnis befinden sich Unity-Anwendungen, die mit MiddleVR entwickelt werden.

aktuell realisiert:
- helloMVR - Hello World, ohne Interaktion
- Fly - Szene mit Zylindern, Default-Fly in MiddleVR. Wir bewegen uns mit Hilfe des Touchpads.
- Naviage - Szene wie in Fly, aber mit Hilfe des MiddleVR-Menus kann die Navigationsmethode verändert werden.
